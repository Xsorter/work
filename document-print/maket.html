Загальні вимоги до макетів

  • 1. Якщо макет не відповідає «Вимогам до макету», вся
  • відповідальність за результат лежить на Замовнику.
  • 2. Кольорова палітра CMYK з напівтонами від 8% до 91%.
  • 3. Роздільна здатність растрових об’єктів 300dpi.
  • 4. Сумарний відсоток заливки (Cyan+Magenta+Yellow+Black) в
  • одній точці не вище 300%.
  • 5. Формати файла pdf у оптимізованій для друку версії.
  • 6. Шрифти необхідно перевести в криві.
  • 7. Поля для порізу 2 мм з кожного боку.
  • 8. Відступ тексту від передбачуваного місця розрізу мінімум 3-4
  • мм.
  • 9. Мінімальна товщина ліній 0.2 pt.
  • 10. Для персоналізації необхідний файл Excel для текстової
  • інформації що персоналізується або набір файлів pdf.
  • 11. Для висічки необхідний векторний файл с контуром ножа.
  • 12. Для тиснення та конгреву необхідний файл з векторним
  • зображенням елементів що тиснуться.
  • 13. Для трафаретного друку необхідний векторний файл без
  • застосування растрових зображень.
  • 14. Максимальні області друку: для цифрового друку 310х440 мм,
  • для офсетного 630х440мм, для трафаретної 900х640 мм.
  • 15. Всі вкладені в файл об’єкти повинні відповідати «Вимоги до
  • макетів».
  • 16. Вбудований в файл профіль кольорів буде видалений.

Типові шаблони

Книги. У друкарні можливо виготовлення книжок у мякої обкладинці з клеєвим блоком (тип КБС) та у твердий обкладинці (тип 7БЦ).

Максимальний обсяг книги у мякому перетені не повинен перебільшувати 600 сторінок (або 300 аркушів). Тверду обкладинку доречно робити на книги обсягом від 120 сторінок.

ШАБЛОНИ ОФОРМЛЕННЯ КНИЖКОВИХ ОБКЛАДИНОК

    КБС

  • Обкладинка КБС (А5) - стандартний формат - обрізний формат 145х205 мм
  • Обкладинка КБС (В5) - журнальний (науковий) формат - обрізний формат 170х240 мм
  • Обклиднка КБС (А4) - великий формат - обрізний формат 205х290 мм
  • Обкладинка КБС (А7) - кішеньковий формат - обрізний формат 100х140 мм

    7БЦ

  • Обкладинка 7БЦ (А5) - стандартний формат - обрізний формат 145х205 мм
  • Обкладинка 7БЦ (В5) - журнальний (науковий) формат - обрізний формат 170х240 мм
  • Обклиднка 7БЦ (А4) - великий формат - обрізний формат 205х290 мм
  • Обкладинка 7БЦ (А7) - кішеньковий формат - обрізний формат 100х140 мм

ПОЛІГРАФІЧНІ ВИРОБИ З ПАПЕРУ

Паперові пакети

  • Стандартні пакети - 250х90х350 мм
  • Маленькі пакети - 190х40х250 мм

Візитки

  • Европейський формат 90х50 мм

Флаер

  • Европейський формат 1/3А4 (99х210 мм)

Буклет А4 на 3 частини

  • Розмір у складеному виді 99х205 мм або 90х185 мм
